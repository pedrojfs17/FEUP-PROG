Os objetivos propostos para o programa foram todos cumpridos:
 - É possível adicionar, remover ou modificar tanto clientes como pacotes, assim como visualizar a informacao de forma formatada. 
 - Os pacotes podem ser filtrados por local, por datas, por disponibilidade, todos os vendidos ou todos os vendidos a um cliente especifico.
 - É possivel fazer a compra de pacotes por um certo cliente.
 - É possivel visualizar o total gasto em pacotes por cada cliente e o total ganho da empresa.

-------------------------------------------------------------------------------------------------------------------

NOTAS:
 - A procura de um cliente, tanto para compra de um pacote, como para visualizar os seus dados, ou para modificar a sua informação, é feita através do seu NIF visto que várias pessoa podem ter o mesmo nome mas o NIF é unico.

 - Para os pacotes, a procura é feita através do seu identificador.

 - Na leitura das datas, o ano é pedido primeiro para verificar se é um ano bissexto, e assim apenas aceitar valores reais para a data no caso de o mês ser fevereiro.

 - Ao filtrar por datas, no meu programa apenas é necessário a viagem estar disponivel num dos dias entre as datas indicadas, para aparecer na lista dos pacotes.
